# Project Steer: Portfolio Rédactrice Web SEO (Premium Edition)

## 1. Vision & Positionnement
- **Cible** : Entreprises haut de gamme, agences de luxe, startups sérieuses.
- **Style Visuel** : "Editorial-meets-Tech". Un mélange entre la rigueur du SEO et l'élégance de la presse écrite de luxe.
- **Objectif** : Transformer l'écriture (immatériel) en une expérience visuelle d'autorité.

## 2. Stack Technique
- **Framework** : Astro 5.x
- **Styling** : Tailwind CSS 4.x
- **UI Components** : Shadcn UI
- **Animations** : Lucide-react (icons), Framer Motion (à ajouter si besoin)
- **Typographie** : 
    - *Heading* : Serif (ex: Playfair Display)
    - *Body* : Sans-Serif (ex: Inter / Geist)

## 3. Direction Artistique
- **Palette** : Minimaliste (Noir, Blanc, Gris très clair, une couleur d'accent profonde).
- **Mise en page** : Grilles bento, typographie oversized, beaucoup d'espace négatif.

## 4. Architecture
- Layout SEO-friendly.
- Components réutilisables.
- Performance maximale.
